# PlatinumRx Assignment

## 📌 Overview

This assignment demonstrates my ability to work with structured data using SQL, perform data analysis using Excel, and implement basic logic using Python.

---

## 🧩 Phase 1 — SQL

### 🔹 Objective

Design and query relational databases for:

* Hotel Management System
* Clinic Management System

### 🔹 Key Concepts Used

* Table creation with primary & foreign keys
* Joins (INNER JOIN)
* Aggregations (SUM, GROUP BY)
* Filtering (WHERE, HAVING)
* Window Functions (RANK, DENSE_RANK)

### 🔹 Highlights

* Calculated billing amounts using quantity × rate
* Identified most/least ordered items per month
* Extracted second highest billing customers using ranking functions

---

## 📊 Phase 2 — Excel

### 🔹 Objective

Analyze ticket and feedback datasets

### 🔹 Key Concepts Used

* VLOOKUP for data mapping using cms_id
* Date comparison using INT()
* Time comparison using HOUR()
* COUNTIFS for conditional aggregation

### 🔹 Highlights

* Linked ticket and feedback datasets
* Identified tickets resolved on same day and same hour
* Generated outlet-wise performance insights

---

## 🐍 Phase 3 — Python

### 🔹 Objective

Implement basic scripts using loops and conditions

### 🔹 Scripts

1. Time Converter
   Converts minutes into hours and minutes format

2. Remove Duplicates
   Removes duplicate characters from a string while preserving order
